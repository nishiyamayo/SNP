SNP-SSP
====================

Sample SSP system.

# Prerequisite
- JDK 1.8 (`brew cask install java`) 
- Optional: 
 - Gradle 2.11

# Building
Ensure that your java version is 1.8

```
$ java -version
java version "1.8.0_45"
```

Build command

```
$ ./gradlew build
```

# Running

```
./gradlew bootRun
```
